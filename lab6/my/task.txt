Даны два строго упорядоченных по возрастанию массива X и Y из n элементов каждый. В массиве X найти все такие элементы X[k], для которых найдутся некоторые элементы из массива Y, удовлетворяющие соотношению: Y[i] < X[k] < Y[i + 1]. Вычислить количество m таких элементов из X и переписать их в первые m элементов массива C.
